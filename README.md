# senakumagames

YouTuber「せなくま」さん（[せなくまチャンネル](https://www.youtube.com/@senakumadesu)）を主人公とした2D・3Dファンゲームのプロジェクトです。

## ドキュメント

- **[GAME_PLAN.md](./GAME_PLAN.md)** — プロジェクト全体の大枠プラン（企画・素材パイプライン・技術スタック・ロードマップ）

## 予定している成果物

1. **せなくまラン 🍟** — 2D横スクロールランナー（Phaser 3）
2. **せなくまアイランド 🏝️** — 3D箱庭探索ゲーム（Three.js + VRM）
3. **Webポータル** — 両ゲームの入口となるページ（GitHub Pagesで公開予定）

> 本プロジェクトは非公式のファンメイド企画です。キャラクター素材の使用は権利者の許諾状況を確認のうえ進めます（詳細は GAME_PLAN.md セクション2参照）。
